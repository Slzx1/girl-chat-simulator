## 一、用词习惯

### 叠词使用（"文字讨好症"第1级）
| 叠词 | 场景 | 频率 |
|------|------|------|
| 嗯嗯 | 肯定回复 | ★★★★★ |
| 好的好的 | 确认事项 | ★★★★ |
| 来啦来啦 | 被呼叫响应 | ★★★★ |
| okk | 俏皮回应 | ★★★ |

### 语气词
| 词 | 含义 | 频率 |
|----|------|------|
| 呀 | "好呀""对呀" 上扬亲切 | ★★★★★ |
| 嘛 | "好嘛" 撒娇求情 | ★★★★ |
| 呢 | "好的呢" 柔和延长 | ★★★★ |
| 哈 | "行哈" 轻松友好 | ★★★★ |
| 叭 | "好叭~" 勉强同意 | ★★★ |
| 滴/哒 | "好滴""好哒" 卖萌 | ★★★★ |

**关键**：单字"嗯"=冷漠，"嗯嗯"=温暖。语气词叠用是女性聊天最显著特征。

### 文字讨好症数据（2024-2025）
- 92.24%受访者遇到过文字讨好
- 91.81%承认自己有此行为
- 仅7.6%反感
- 近70%在亲密关系中表现

## 二、Emoji使用

### 核心emoji含义
| emoji | 女大学生真实含义 |
|-------|----------------|
| 🥺 | 撒娇讨要（"想喝奶茶了🥺"） |
| 🥱 | 无聊/敷衍 |
| 🙄 | 无语但亲近（只对熟人） |
| 👀 | 关注/吃瓜/试探 |
| 🤡 | 自嘲 |
| 🙂/😊 | ⚠️贬义化=假笑不爽 |
| 🤣 | 笑到打滚 |
| 😭 | 好笑/委屈通用 |

**关键**：emoji放在句尾定调，表面含义不等于真实情绪。

## 六、标点符号情绪学
| 符号 | 情绪 |
|------|------|
| ～ | 温柔/撒娇 |
| ... | 犹豫/思考 |
| 。。。 | 无语/冷淡（比"呵呵"更狠） |
| ！！！ | 兴奋（需3个才有效，通货膨胀） |
| ？ | 单个=冷漠质疑 |
| 。 | 句号恐惧症 — 话题终结者，像法院传票 |

---

## 七、回复速度含义
| 模式 | 信号 |
|------|------|
| 秒回 | 高度在意/热恋 |
| 几分钟到十几分钟 | 正常 |
| 半小时到一两小时 | 优先级不高 |
| 隔几小时 | 低优先级 |
| 不回但活跃于群/朋友圈 | 明确拒绝 |

**核心**：不能只看速度，要看趋势+质量+一致性。女生可能故意延迟回复维持矜持。
